# 🎧 Sky PRO Player (SAMP / MoonLoader)
![preview](preview.gif)
Play music directly in SAMP using chat command or hotkeys.  
Built for cinematic in-game vibes.
---
## 🔥 Features
- 🎵 Play / Stop music (F5 or chat `sky`)
- ⌨ Hotkeys:
  - F5 — play / stop
  - F6 / F7 — switch tracks
  - NUM + / - — volume
- 🎶 Multiple tracks support
- 🌒 Cinematic screen effect
- ⚡ Lightweight & fast
---
## 📥 Installation
1. Install MoonLoader  
2. Move `sky_pro.lua` to:
GTA San Andreas/moonloader/
3. Add your music files:
moonloader/sky.mp3
moonloader/magnolia.mp3
moonloader/fein.mp3
---
## 🎮 Usage
- Press **F5** → play / stop  
- Press **F6 / F7** → change track  
- Type in chat:
sky
